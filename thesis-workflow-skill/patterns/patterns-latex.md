# patterns-latex: 北航学位论文 LaTeX 模板定制约束 

当处理用户项目中的 `.tex` 源码，或使用《北航硕博士学位论文模版-LaTeX 4.1.0》时，必须恪守北航模板和北航撰写规范。`graduate-thesis-polish-and-write-skill` 仅作为语言风格参考，不能覆盖本文件中的格式、排版、宏命令和编译要求。

---

## L. 排版与编译定制限制（适配 BUAA v4.1.0 等）

### L0. 格式规则优先级
- 北航模板 4.1.0、`buaa.cls`、模板 README、`main.tex` 示例和北航撰写规范是最高格式依据。
- 如果外校 polish 规则、范例论文习惯或通用 LaTeX 建议与北航模板冲突，一律以北航模板为准。
- 不要为了“去 AI 味”改动页边距、字号、行距、页眉页脚、目录缩进、封面/题名页、摘要关键词、附录编号、参考文献样式等格式项。

### L1. 编译引擎
- 必须使用 **XeLaTeX** 进行构建与编译。

### L2. 参考文献样式 (Bibliography)
- 国标参考文献样式要求：必须使用 `\Bib{def/GBT7714-2015.bst}{ref.bib}` 或 `\Bib{def/GBT7714-2015-NoWarning.bst}{ref.bib}`。
- （新版本中如若缺失出版社地点信息，NoWarning 可消隐缺失警告符号 `[s.n.]` 和 `[S.L.]`）。

### L3. 符号和缩略语
- 模板将符号和缩略语作了职能拆分：
  - 使用 `\Signs` 来输入“主要符号表”
  - 使用 `\Abbreviations` 来输入“缩略语说明”
- 不要沿用旧模板或外校模板把二者合并为一个泛化的“符号说明”部分。

### L4. 图表相关 (Figures & Tables)
- **子图宏包约束**：模板采用现代规范，**必须**使用 `subcaption` 宏包，废弃了原有的 `subfigure` 等陈旧命令。子图排版请采用规范的 `\begin{subfigure}...\end{subfigure}` 形式。
- **三线表粗细约束**：若需加粗三线表的边界，**必须使用** `\toprule[]`, `\midrule[]`, 以及 `\bottomrule[]` 命令。
- 图表编号格式、题注距离和目录表现由北航模板控制，不按外校 skill 的图表格式习惯改写。
  
### L5. 公式排版细节 (Equation Spacing)
- **绝对禁止空行尾随**：在使用 `equation` 等数学环境写完公式后，**请不要在公式环境之后加入任何额外的空行**，这在 BUAA v4.1.0 模板中会导致公式与下一行正文文字的页间距过大。（此条覆盖通用的 C8 规则中关于强制加空行的习惯）。

### L6. 特殊命令说明
- **无页码空白页**：如需为了占位排版右侧起页，手动插入无页码的空白页，必须使用模板提供的特殊命令 `\emptypage`（代替老旧模板中的 `\beginright`）。

### L7. 模板结构保护
- 保留 `\documentclass[...] {def/buaa}` 的模板结构，通过文档类选项设置论文类型、密级、打印设置、系统类型、标题长短和学科类型。
- 保留模板提供的封面、题名页、摘要、关键词、图表目录、符号表、缩略语、成果、致谢、附录等结构命令。
- 添加宏包前先检查 `buaa.cls` 是否已经内嵌；避免重复加载或引入与模板冲突的格式宏包。
- 参考文献不显示时，优先运行 BibTeX/参考文献步骤并重新 XeLaTeX 编译，不随意改 `\cite` 或 `\Bib` 结构。
