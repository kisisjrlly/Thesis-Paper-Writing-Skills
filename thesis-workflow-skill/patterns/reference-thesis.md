# reference-thesis: 范例论文配置

本 skill 依赖**用户指定的范例论文**作为风格基线。不同学科、不同学校、不同导师的毕业论文风格差异很大，没有一本"通用范例"能覆盖所有情况。由你（用户）来决定参照哪一本。

---

## 怎么配置（两种方式，二选一）

### 方式 A：对话里直接告诉 skill（推荐）

不用打开本文件。直接在对话里说：

```
参考论文是 /home/user/docs/senior.pdf
```

或者

```
对照 /path/to/A.pdf 和 /path/to/B.pdf 这两本。
```

skill 会自动把信息写到下面的"我的范例论文"区块里，下次不用再说。

### 方式 B：手动编辑本文件

在下面"我的范例论文"区块里按模板填也行。**最关键的字段是文件路径**，其他能填就填、没有就留空。

## skill 如何使用这些范例

为了控制 token 成本，skill 采用"指纹"机制：

1. **首次配置范例论文时**（通过对话或手动填本文件），skill 会 Read 一次原文，提取一个约 5-10 KB 的风格指纹，存到 `reference-thesis-fingerprint.md`
2. **后续所有对话**默认只加载指纹文件，不再碰原文，大幅省 token
3. **指纹不够用时**（需要对照具体表达、或有深度风格对齐需求），skill 会先用 Grep 搜原文关键片段，再不够才局部 Read 原文某章
4. **范例论文换了**：skill 会自动重新生成指纹

指纹的具体 schema 见 SKILL.md 的"风格指纹"章节。你可以直接打开 `reference-thesis-fingerprint.md` 查看提取出的样例，不满意可以要求 skill 重建。

---

## 没有范例论文时

skill 会提示这个状态，并询问是否以"纯规则模式"继续。

纯规则模式下会失去：具体风格锚点、公式复杂度参考、章节结构参考、图表样式参考。能用，但效果打折。

---

## 怎么找范例论文

- 本校图书馆数据库通常能下载近几年的优秀硕博论文
- CNKI 按学科筛选硕士学位论文，按下载量或引用排序
- 师兄师姐推荐
- 你自己读过觉得"这本写得真好"的同方向论文

选的时候看两点：
- **语言**：读起来顺，没有 AI 味，也不是老干部体
- **结构**：目录清晰、章节引言扎实、小结能回应问题
